Labeling

This is where you can do some labeling of your dataset if necessary, but usage will depend on your specific model or problem. For example:

  • a model like SAM3 can generate a segmentation mask based on a provided bounding box
  • models may support preannotating data
  • some models allow you to create a new labeling project with most things preconfigured
  • otherwise you might need to follow upstream documentation for additional info on configuration

It uses Label Studio, a web application for labeling with support for a wide variety of tasks, while making some adjustments for a more seamless experience in the toolbox.
